Project Description:
This project is created as part of my DevOps internship to practice Git, GitHub workflow, branching strategies, pull requests, issue tracking, GitHub Actions, and documentation. It contains all assignments completed during the internship.

How to Contribute:
First fork this repository to your GitHub account.
Clone your fork into your system.
Switch to the develop branch and pull the latest code.
Create a new branch for your work.
Make your changes and test them.
Commit your changes with a clear message.
Push your branch to your fork.
Open a pull request to the develop branch.
Wait for review and update the PR if any changes are requested.
